ZanLive


一个简洁好用的直播+IM系统.

## 目录内容

```text
├─ README.md
├─ doc/  					# 项目文档、SQL、接口说明、设计资料
├─ zanlive-client-uniapp/   # App客户端
├─ zanlive-im-web/			# 桌面客户端
├─ zanlive-publish/			# 静态下载发布页
└─ zanlive-server-java/     # 服务端
 
```
